# Quick Installation Guide

## Step 1: Build
```bash
yarn
yarn build
```

## Step 2: Load in Chrome
1. Open `chrome://extensions/`
2. Enable **Developer mode** (top right)
3. Click **Load unpacked**
4. Select the `dist` folder

Done! The extension will work on Google Calendar pages.

## No Publishing Required
You can use the extension without publishing it to Chrome Web Store. Just load it as an unpacked extension for personal use.
